# QoL Roadmap

Goal: make the control panel a daily workbench, not just a VM launcher.

## Next

1. Quick links per agent:
   - backend app;
   - Hasura;
   - VS Code Remote.
2. Diff workflow:
   - file list;
   - rendered patch;
   - live refresh while an agent works.
3. Fork agent:
   - clone existing agent to next free number;
   - block or warn when Codex is working.
4. Stop controls:
   - stop selected;
   - stop all idle.
5. Resource visibility:
   - rough memory per agent;
   - high-pressure warnings;
   - cleanup actions.

## Hotkeys Later

Use app-level hotkeys only outside terminal/input focus.

Candidate keys: `j/k`, `enter`, `n`, `s`, `d`, `w`, `b`, `c`, `x`, `?`, `esc`.
